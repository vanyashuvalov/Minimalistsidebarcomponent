## 🚀 Как использовать

### Метод 1: Автоматический деплой (рекомендуется)

```bash
# 1. Пушим код
git add .
git commit -m "Initial commit"
git push origin main

# 2. Включаем GitHub Pages
# Settings → Pages → Source: GitHub Actions

# 3. Готово! Сайт автоматически задеплоится
```

```bash
# 1. Устанавливаем зависимости
npm install

# 2. Деплоим
npm run deploy

# 3. Включаем GitHub Pages
# Settings → Pages → Source: gh-pages branch
```

## ❓ Частые вопросы

### Сайт не загружается

**Решение:**
1. Проверьте Settings → Pages → Source = GitHub Actions
2. Очистите кеш браузера (Ctrl+Shift+R)
3. Подождите 2-3 минуты после деплоя

### CSS не работает

**Решение:**
1. Проверьте `vite.config.ts` - должно быть `base: './'`
2. Убедитесь, что файл `.nojekyll` есть

### Actions падает с ошибкой

**Решение:**
1. Проверьте логи в Actions
2. Убедитесь, что package.json валидный
3. Проверьте, что используете Node.js 20
